PlaceholderLikelihood.analytic_ellipse_loglike — Function

Analytic ellipse log-likelihood has no knowledge of lower and upper bounds on parameters. Hence profiles generated by optimising out the nuisance parameters for a given interest parameter may look different if the analytical profile enters space where a bound would be active. Pushing forward from these confidence bounds may be infeasible - if analytical profile has entered a space where a parameter bound is active.
